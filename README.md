发现：int_commit和fr_commit并不是严格的8倍关系（理论上），原因是：原始的tensor数据大部分绝对值都很小，导致高位存在很多的0，所以int_commit快一些。如果用随机初始的int数据就不会太快了，那样的话还是fr的速度会更快（时间是int承诺的不到8倍）。
